# Button

Awesome button, used to take actions in forms, dialogs and navigation.
Provides seven different color variants: `primary`, `secondary`, `info`, `danger`, `success`, `ghost`, `card`.

The Button component is based on [React Aria](https://react-spectrum.adobe.com/react-aria/getting-started.html) unstyled and accessible [button component](https://react-spectrum.adobe.com/react-aria/Button.html).

### Variants

Buttons can have different variant colors by setting the `variant` prop. The default variant is `primary`

<div class="custom-canvas">
  <Story of={ButtonStories.Primary} />
  <Story of={ButtonStories.Secondary} />
  <Story of={ButtonStories.Info} />
  <Story of={ButtonStories.Danger} />
  <Story of={ButtonStories.Success} />
  <Story of={ButtonStories.Ghost} />
  <Story of={ButtonStories.Card} />
</div>

<Source dark code={`
<Button variant="primary">primary</Button>
<Button variant="secondary">secondary</Button>
<Button variant="info">info</Button>
<Button variant="danger">danger</Button>
<Button variant="success">success</Button>
<Button variant="ghost">ghost</Button>
<Button variant="card">card</Button>
`} />

### Types

Specifies the button type when used in an HTML form. Specify `button`, `submit` or `reset`. The default type is `button`.


### Sizes

Buttons can have different sizes by setting the `size` prop. The default size is `md`.

<div class="custom-canvas">
  <Story of={ButtonStories.Small} />
  <Story of={ButtonStories.Medium} />
  <Story of={ButtonStories.Large} />
</div>

<Source dark code={`
<Button size="sm">small</Button>
<Button size="md">medium</Button>
<Button size="lg">large</Button>
`} />

### Disabled

Specifies a non interactive button with `isDisabled` prop to render a button with disabled state.

<div class="custom-canvas">
  <Story of={ButtonStories.Disabled} />
</div>

<Source dark code={`

`} />

### Full width

Create a full width buttons by setting the `isFull` prop.

<div class="custom-canvas full">
  <Story of={ButtonStories.FullWidth} />

<Source dark code={`

### Properties

- **`<Button>`** properties:

<Markdown>
{`
| prop    | type     | default    | description 
|---------|----------|------------|
| variant | string   | 'primary'  | Applies color variant to the button. Available variants: 'primary', 'secondary', 'info', 'danger', 'success', 'ghost', 'card'.
| size    | string   | 'md'       | Applies the size of the button. Available sizes: 'sm', 'md', 'lg'.
| isFull  | boolean  | false      | Applies 100% to the button width.
`}
</Markdown>

To see all button properties, see [React Aria button properties](https://react-spectrum.adobe.com/react-aria/Button.html#props).
