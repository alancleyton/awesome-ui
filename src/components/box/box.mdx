# Overview

Awesome Box, used to build blocks and create responsive layouts, also used to grouping other components.

### Usage

By default the box component renders a `<div>` HTML element, but you can switch to another valid HTML element using the `is` prop.

<Canvas>
  <Story of={BoxStories.Default} />
</Canvas>

### is prop

Use the `is` prop to change the HTML element render.

<Canvas>
  <Story of={BoxStories.Span} />
