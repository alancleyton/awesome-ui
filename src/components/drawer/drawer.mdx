### Drawer placement

The Drawer component sits off-screen until triggered by the user, it can appear from any corner of the side.
Use the `placement` prop to specify which side of the screen the Drawer should appears.
Placements can be `top`, `right`, `bottom`, `left`. The default size is `left`

<Source dark code={`
const DrawerPlacementsExample = () => {
  const [isOpen, setIsOpen] = useState(false);
  const [placement, setPlacement] = useState('left');

  const openDrawer = () => setIsOpen(true);

  const closeDrawer = () => setIsOpen(false);

  const placements = ['top', 'right', 'bottom', 'left'];

  const changePlacement = (newPlacement) => {
    setPlacement(newPlacement);
    openDrawer();
  };

  <>
    {placements.map((placement) => (
        <Button
          onPress={() => changePlacement(placement)}
          key={placement}
        >
          {placement} drawer
        </Button>
    ))};

    <Drawer placement={placement} open={isOpen} onClose={closeDrawer}>
      <Drawer.Content>
        <div className="p-5 h-full bg-white">
          <p>drawer content</p>
          <Button onPress={handleCloseDrawer}>close drawer</Button>
        </div>
      </Drawer.Content>
    </Drawer>
  </>
};
`} />

### Drawer sizes

Drawer content panel can have different sizes by setting the `size` prop.
The size prop can receive `sm`, `md`, `lg` and `full` values. The default size is `md`.
