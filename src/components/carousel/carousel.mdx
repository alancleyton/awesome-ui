# Carousel

Awesome carousel, a slider used to displays a series of content (images, text, videos, etc.) in a rotating slideshow.

### Table of Contents

- [Import](#import)
- [Usage](#usage)
- [Controled](#controled)

`} />

### Properties

- **`<Carousel>`** properties:

<Markdown>
{`
| prop        | type      | default    | description
|-------------|-----------|------------|
| interval    | number    | null       | Delay time to automatically cycling an item.
| activeIndex | number    | 0          | Renders the carousel with active item.
| nextIcon    | ReactNode | undefined  | Change the default button icon for the next control.
| prevIcon    | ReactNode | undefined  | Change the default button icon for the prev control.
`}
</Markdown>
