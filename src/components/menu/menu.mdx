# Navbar

Awesome Menu, used to display a list of options that a user can choose.
The Menu component is based on [React Aria](https://react-spectrum.adobe.com/react-aria/getting-started.html) unstyled and accessible [menu component](https://react-spectrum.adobe.com/react-aria/Menu.html).

### Usage

Example of the `<Menu>` component:

<div class="custom-canvas">
  <Story of={MenuStories.BasicMenu} />
</div>

<Source dark code={`
import { Menu } from '@alancleyton67/awesome-ui';

export default () => {
  return (
    <Menu label="Open menu" onAction={(action) => 'selected action: ' + action}>
      <Menu.Item id="myAccount">My Account</Menu.Item>
      <Menu.Item id="updateAccount">Update Account</Menu.Item>
      <Menu.Item id="logout">Logout</Menu.Item>
    </Menu>
  )
}
`} />

### Menu sections

To group related menu `items`, you can use `<Menu.Section>` component and pass it a `title` for the group name.

<div class="custom-canvas">
  <Story of={MenuStories.MenuWithSections} />
</div>

<Source dark code={`
import { Menu } from '@alancleyton67/awesome-ui';

export default () => {
  return (
    <Menu label="Profile">
      <Menu.Section title="profile">
        <Menu.Item>Update Account</Menu.Item>
        <Menu.Item>Account Dashboard</Menu.Item>
        <Menu.Item>Appointments</Menu.Item>
      </Menu.Section>
      <Menu.Section title="actions">
        <Menu.Item>Logout</Menu.Item>
        <Menu.Item>Delete Account</Menu.Item>
      </Menu.Section>
    </Menu>
  )
}
`} />

### Menu separators

You can also group menu `items` using separators, use `<Menu.Separator>` to add a horizontal `divider` between menu items.

<div class="custom-canvas">
  <Story of={MenuStories.MenuWithSeparator} />
</div>

<Source dark code={`
import { Menu } from '@alancleyton67/awesome-ui';

export default () => {
  return (
    <Menu label="File">
      <Menu.Item>New File</Menu.Item>
      <Menu.Item>New Document</Menu.Item>
      <Menu.Separator />
      <Menu.Item>Open File</Menu.Item>
    </Menu>
  )
}
`} />

### Disabled items

You can have disabled items by passing a list of disabled items to `disabledKeys` prop in `<Menu>` component.
An item is disabled if its id exists in `disabledKey` prop, so you have to set an id to the `<Menu.Item>`.

<div class="custom-canvas">
  <Story of={MenuStories.MenuWithDisabledItems} />
</div>

<Source dark code={`
import { Menu } from '@alancleyton67/awesome-ui';

export default () => {
  return (
    <Menu label="Fuits" disabledKeys={['cranberry']}>
      <Menu.Item id="passionFruit">Passion Fruit</Menu.Item>
      <Menu.Item id="banana">Banana</Menu.Item>
      <Menu.Item id="cranberry">Cranberry</Menu.Item>
    </Menu>
  )
}
`} />

### Properties

- **`<Menu>`** properties:

<Markdown>
{`
| prop    | type    | default   | description
|---------|------- -|-----------|
| label   | string  |           | Label for menu trigger button. It can be a string or JSX element.
| isOpen  | boolean | false     | If true, the menu is shown.
| variant | string  | 'primary' | Applies color variant to the trigger button. Available variants: 'primary', 'secondary', 'info', 'danger', 'success', 'ghost', 'card'.
| size    | string  | 'md'      | Applies the size of the trigger button. Available sizes: 'sm', 'md', 'lg'.
`}
</Markdown>
