### Width

Create full width inputs by setting the `isFull` prop in `<Input.Root>`.
The default width of the `<Input>` is `auto`.

<div class="custom-canvas">
  <Story of={InputStories.FullWidthInput} />
</div>

<Source dark code={`
<Input.Root isFull>
  <Input.Field placeholder='Full input' />
</Input.Root>
`} />

### Addons and Elements

Add addons and elements to the left and right of the `<Input>` component.
You can use the `<Input.RightAddon>`, `<Input.LeftAddon>`, `<Input.RightElement>`, `<Input.LeftElement>`
with the `<Input.Group>` to help with addons and elements position.

<div class="custom-canvas">
  <Story of={InputStories.RightAddonInput} />
</div>

<Source dark code={`
<Input.Root>
  <Input.Group addonRight>
    <Input.Field placeholder='Addon right input' />
    <Input.RightAddon>@</Input.RightAddon>
  </Input.Group>
</Input.Root>
`} />

<div class="custom-canvas">

### Properties

- **`<Input.Root>`** properties:

<Markdown>
{`
| prop     | type     | default     | description 
|----------|----------|-------------|
| variant  | string   | 'secondary' | Applies color variant to the input. Available variants: 'primary', 'secondary', 'info', 'danger', 'success'.
| size     | string   | 'md'        | Applies the size of the input. Available sizes: 'sm', 'md', 'lg'.
| isFull   | boolean  | false       | Applies 100% to the input width.
`}
</Markdown>

- **`<Input.Group>`** properties:

<Markdown>
{`
| prop         | type    | default | description 
|--------------|---------|---------|
| addonRight   | boolean | false   | Applies the right addon to the input field.
| addonleft    | boolean | false   | Applies the left addon to the input field.
| elementRight | boolean | false   | Applies the right element to the input field.
| elementLeft  | boolean | false   | Applies the left element to the input field.

`}
</Markdown>
