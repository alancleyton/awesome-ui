# Overview

Awesome Navbar, used to help users navigate to other pages. Can contain links to easily access to the pages or sections.

## Table of Contents

- [Properties](#properties)

## Properties

- **`<Navbar>`** properties:

<Markdown>
{`
| prop | type | default | description 
|----------|----------|----------|
| variant | string | 'primary' | Applies background color to the navbar. Available variants: 'primary', 'secondary'.
| size    | string | 'md'      | Applies the size of the navbar. Available sizes: 'sm', 'md', 'lg'.
`}
</Markdown>

- **`<Navbar.Toggler>`** properties:

<Markdown>
{`
| prop | type | default | description 
|----------|----------|----------|
| alignment | string  | 'left' | Determines the horizontal position of the navigation toggler. Available alignments: 'left', 'right', 'center'.
`}
</Markdown>

- **`<Navbar.Nav>`** properties:

<Markdown>
{`
| prop | type | default | description 
|----------|----------|----------|
| alignment | string  | 'left' | Determines the horizontal position of the navigation items. Available alignments: 'left', 'right', 'center'.
| filled    | boolean | false  | Applies 100% to the navbar width. 
| collapse  | boolean | false  | Hide items for small screens and show them on the desktop screen.
`}
</Markdown>

## Usage

The `<Navbar>` component is composed by others components to compose the navigation bar. Use the `<Navbar.Brand>`, `<Navbar.Toggler>`,
`<Navbar.TogglerBurger>`, `<Navbar.Nav>`, `<Navbar.NavItems>`, `<Navbar.NavItem>` and `<Navbar.NavLink>` components.

Available composable components in the Navbar component:
- The `<Navbar>` is a wrapper to composed components.
- The `<Navbar.Brand>` component renders a `<div>` by default, to use as a link, set the `href` prop.
- The `<Navbar.Toggler>` is a button that shows up when the screen size is small, and the navigation links are often hidden.
- The `<Navbar.TogglerBurger>` is a toggler button burger icon.
- The `<Navbar.Nav>` is a content wrapper that can be horizontal positioned into the navbar.
- The `<Navbar.NavItems>` is a list for the navigation items.
- The `<Navbar.NavItems>` is a item of the navigation items.
- The `<Navbar.NavLink>` is a navigation link.

## Collapse

The `<Navbar.Nav>` component has the `collapse` prop, which is used for responsive behaviors.
Hide large amount of items for small screens and show them on the desktop screen by setting the collapse prop.
The default collapse prop value is `false`.

## Color variants
