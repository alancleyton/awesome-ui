# Getting Started
Awesome UI provides ready-to-use, accessible, reusable, and composable
React UI components based on the [Tailwind CSS](https://tailwindcss.com/) framework utility classes.

## Requirements
Made sure to have the correct versions installed in your project.
* [React](https://react.dev) v18 is required

## Installation
To use Awesome UI in your project, you need to install the package, with npm or yarn, run one of the following commands in your terminal:

```sh
# with Yarn
$ yarn add @alancleyton67/awesome-ui

# with npm
$ npm i @alancleyton67/awesome-ui
```

## Usage

To start using the components, on your application entry point, import the CSS file.

```jsx
// Do this at the root of your application
import '@alancleyton67/awesome-ui/dist/index.css';
```

Start using the components in your application

```jsx
import { Button } from '@alancleyton67/awesome-ui';

const App = () => {
  const onPress = () => alert('On pressed!')

  <div>
    <Button variant="info" onPress={onPress}>
      info button
    </Button>
  </div>
};

export default App;
```
